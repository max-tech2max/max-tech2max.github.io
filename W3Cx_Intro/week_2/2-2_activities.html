Week 2.2 Activities

Activity 1

Find the list of supported attributes for the <area> tag.

In HTML5 the <area> tag has the following Attributes:

  • alt
  • coords
  • download
  • href
  • hreflang
  • media
  • nohref
  • rel
  • shape
  • target
  • type

The area tag also supports the global and event attributes.

Activity 3

Does an attribute called 'src' exist? What is the purpose of this attribute and what are the elements it can be applied to?

The 'src' attribute is used in conjunction with tags such as <audio>, <img>, <script>, & <video> (among others).

The required 'src' attribute specifies the absolute or relative URL of the item.
